動作選擇器(Action Selector)

內容包含動作選擇器(ActionName、NonAction、ActionVerbs)，敘述如下：

  • ActionName：用設定動作別名來取代原有名稱，使用方式在動作宣告之前設定 [ActionName("別名")] 如下圖(請思考如何修正第三張圖錯誤)：

  • NonAction：用設定僅供內部程式呼叫的方法，NonAction設定動作無法從網址列呼叫來執行，使用方式在動作宣告之前設定 [NonAction] 。
  • ActionVerbs：用於動作路由設定包含 HttpGet、HttpPost、HttpPut、HttpDelete ，例如同樣的動作名稱，但是要因應不同的HTTP模式或是代入參數數目執行不同的動作。 
    public IActionResult Privacy()
{
    return View();
}

[HttpGet("{id:int}")]
public IActionResult Privacy(int id)
{
    return View("Index");
}
