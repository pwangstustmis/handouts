CSS簡介

了解常用HTML標簽後，使用者會希望網頁能有自訂版面規劃如字型、顏色、位置等，因此導入CSS(Cascading Style Sheet)，學習CSS除了必須了解CSS屬性意義及其設定方式，更重要的是樣式的套用對象(selector)，除此之外，請注意CSS具傳遞性質，對某一對象設定樣式時，所有包含於這個對象中的內容，預設就會套用該對象樣式。本單元僅介紹常用CSS屬性，詳細的介紹，請參考W3Schools CSS。

CSS設定語法

CSS設定包含行內(inline)、內部(internal)與external(外部)三種定義方式。行內(inline)是指將樣式設定於HTML標籤style屬性中，設定方式為 屬性:樣式值; ，設定可包含多個屬性。內部(internal)是將樣式定義於HTML檔檔頭(head)位置內，並包含於HTML style標籤中，設定方式為 設定對象 {屬性:樣式值;} ，設定可包含多個屬性。external(外部)是將樣式存於副檔名為 .css 的獨立檔案，並在網頁檔頭(head)位置內利用HTML link標籤載入對應的樣式檔。接下來以行內(inline)、內部(internal)與external(外部)三種定義方式來定義選擇對象為段落(p)，樣式為紅色字體、大小12pt。

段落紅色字體、大小12pt

行內(inline)CSS設定：

<p style="color:#f00;font-size:12pt;">段落紅色字體、大小12pt</p>
內部(internal)CSS設定[設定對象為段落標籤p]：

<head>
  <style>
    p {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p>段落紅色字體、大小12pt</p>
外部(external)CSS設定，將內部設定中段落p樣式存於外部檔案假設為style.css：

<head>
  <link rel="stylesheet" href="style.css">
</head>
...
<p>段落紅色字體、大小12pt</p>

練習：請建立外部檔案style.css並測試樣式套用成功。

設定對象-簡單對象

CSS樣式設定如同先前HTML標籤，使用者必須了解各個CSS樣式屬性設定方式，但是在介紹常用CSS屬性之前，使用者必須先了解樣式套用對象的設定，對象設定包簡單(simple)、混合(combinator)、虛擬類別(pseudo classes)與屬性選擇(attribute selector)四大類，這裡僅對每一類常用項目作介紹，其他設定請參考連結網頁。簡單對象包含HTML標籤名稱、id名稱、自訂類別、全部對象與多個對象。

  • HTML標籤名稱會將樣式套用於網頁所有設定對象標籤，例如在CSS語法中段落標籤p設定。
  • 網頁標籤可以設定不重複id屬性，id屬性用於識別各個標籤，CSS設定id屬性對象必須搭配 # 來標示指定id標籤。
  • 自訂類別為了讓樣式設定更具彈性，使用者在HTML標籤透過屬性 class 即可套用樣式。
  • 全部對象使用 * 代表網頁全部標籤屬。
  • 多個對象使用 , 來隔開多個選擇對象。以下範例均以內部(internal)CSS設定表示。
HTML標籤名稱CSS設定：

<head>
  <style>
    p {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p>段落紅色字體、大小12pt</p>
id屬性CSS設定：

<head>
  <style>
    #pr12 {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p id="pr12">段落紅色字體、大小12pt</p>
自訂類別CSS設定：

<head>
  <style>
    .pr12 {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p class="pr12">段落紅色字體、大小12pt</p>
全部對象CSS設定：

<head>
  <style>
    * {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p>段落紅色字體、大小12pt</p>
多個對象CSS設定：

<head>
  <style>
    p,h1 {
      color:#f00;
      font-size:12pt;
    }
  </style>
</head>
...
<p>段落紅色字體、大小12pt</p>
<h1>段落紅色字體、大小12pt</h1>

練習2：請在網頁中設定一大標題(h1)的id名稱為 myh1 ，並使用CSS id屬性對象設定其字體大小設為20pt、橘色。

練習3：請修改練習2為自訂類別CSS設定，設定一CSS類別為 myh1 ，字體大小設為20pt、橘色並套用於任一HTML標籤。

設定對象-混合對象

混合對象藉由空白、>、+與~來設定樣式套用的關係。
對象1 對象2 對象1空白對象2，代表所有包含於對象1的對象2套用該樣式。
對象1 > 對象2 當對象2為對象1小孩時套用該樣式。
對象1 + 對象2 對象2為對象1立即相鄰的兄弟套用該樣式。
對象1 ~ 對象2 對象2為對象1之後的兄弟套用該樣式。

      <p>隸屬段落p小孩，隸屬div小孩(p)-小孩(p)</p>

  </div>
</body>

    div p {

設定對象-虛擬類別

虛擬類別用於定義當類別處於某些狀態時套用的CSS樣式，使用方式為 類別名稱:虛擬類別 ，常用虛擬類別包含：

  • active 用於設定點選時CSS類別類別。
  • checked 用於設定核取時CSS類別類別。
  • focus 用於設定文字方塊點選時CSS類別類別。
  • hover 用於設定滑鼠移上時CSS類別類別。
  • link 用於設定連結標籤a未曾拜訪過CSS類別類別。
  • visited 用於設定連結標籤a拜訪過點選時CSS類別類別。

CSS屬性設定-顏色

CSS顏色設定包含前景(字)與背景。前景顏色屬性名稱為 color ，顏色值設定可以用 #紅綠藍 其中紅綠藍分別代表紅色、綠色與藍色，設定值用16進位制(00~ff)，或是利用 rgb(紅,綠,藍) 與 rgba(紅,綠,藍,透明度) 來設定，使用 rgb(紅,綠,藍) 各個顏色值為0到255，a代表透明度，值為0到1，0代表透明(完全看不到)，1代表完全不透明(完全看得到)。背景屬性設定包含背景顏色 background-color 、背景圖片 background-image 、圖片重複方式 background-repeat 、背景黏貼方式 background-attachment 、背景位置 background-position 與 background ，而 background 屬性是用一個屬性來整合表示先前個別背景屬性。背景顏色屬性 background-color 同前景顏色設定，背景圖片 background-image 設定為 url("圖片檔案位置") ， 圖片重複方式 background-repeat 預設為水平與垂直重複，使用者可以設定僅水平重複(repeat-x)、僅垂直重複(repeat-y)或不重複(no-repeat)。背景黏貼方式 background-attachment 值包含固定(fixed)與捲動(scroll)，設為固定則背景圖不會隨著視窗捲動而改變位置，捲動則背景圖會隨著視窗捲動而改變位置。背景位置 background-position 常用值包含左上(left top)、左中(left center)、左下(left bottom)、中上(center top)、中中(center center)、中下(center bottom)、右上(right top)、右中(right center)、右下(right bottom)。background 屬性則是把先前屬性設定以一行表示。

CSS屬性設定-字型

CSS字型常用設定包含字型(font-family)、大小(font-size)、字樣式(font-style)與字粗細(font-weight)。

字型(font-family)設定為 font-family:字型 ，預設字型包含 Sans-serif 、 Arial 、 Verdana 、 Helvetica 、 Tahoma 、 "Trebuchet MS" 、 "Times New Roman" 、 Georgia 、 Garamond 、 "Courier New" 與 "Brush Script MT"。設定方式為 font-family: 字型名稱 ，多個字型則以逗點隔開，請注意字型設定在最後一般會包含 Sans-serif ，目的是為確保當先前字型無法在入時，瀏覽器可以用預設字型顯示。除了預設字型外，使用者也可以載入外部字型，以Google Fonts為例，載入方式如下：
在網頁擋頭位置加入 <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Google字型名稱"> 後，接下來在字型(font-family)中套用Google字型名稱即可。使用者可參閱Ggoole字型。

常用字型大小(font-size)以像素點(px)、點(pt)、em約16px大小(em)及vw代表現在視窗寬度值為0到100(100)，其中vm會隨著視窗大小改變字體大小。

常用字型樣式(font-style)包含正常(normal)或斜體(italic)。

常用字型粗細(font-weight)值從100到900，400 為正常字體粗細，700 為粗體字粗細，使用者可自行設定粗細值。

CSS屬性設定-邊界與內距

CSS字型邊界(margin)用於設定標籤與標籤之間的距離包含上(margin-top)、下(margin-bottom)、左(margin-left)、右(margin-right)，如果要同時設定4邊邊界可以用 margin 來進行設定。內距(padding)用於設定標籤內容上下左右的間距包含上(padding-top)、下(padding-bottom)、左(padding-left)、右(padding-right)，如果要同時設定4邊內距可以用 padding 來進行設定。邊界與內距值可以是正負值，間距大小設定同字型大小。

CSS屬性設定-邊框

CSS字型邊框(border)用於設定邊框，設定方式為 邊框屬性: 邊框粗細 邊框顏色 邊框樣式 ，粗細設定與自性大小設定相同，邊框顏色與顏色設定相同，邊框樣式包含圓點(dotted)、虛線(dashed)、實線(solid)與雙線(double)，邊框屬性包含上(border-top)、下(border-bottom)、左(border-left)、右(border-right)，如果要同時設定4邊邊框可以用 border 來進行設定。圓角邊框會搭配border-radius屬性設定4個角的半徑值大小。

CSS屬性設定-對齊
