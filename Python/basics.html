Python語法

Python檔案的副檔名為.py，縮排在Python代表程式區塊，在Python中程式區塊不像一般程式語言使用 { } ，相反地使用縮排的方式來代表程式區塊，具有相同間距的程式屬於同一區塊，間距的大小並沒有限制，但是不同的程式區塊最少要間距1個空白。以下徒為例，我燜可以看到程式 print(msg) 出現錯誤，由於它屬於條件 if 的程式區塊，而它與對應 if 命令的啟示位置相同，程式註解使用 # 如下圖。

變數設定

變數的目的用於儲存輸入資料或計算結果，Python變數名稱可由英文字或底線符號(_)開始，同時變數名稱大小寫英文字視為不同的變數，不同於強形態語言，Python變數不須宣告儲存類型，同時變數資料類型會隨儲存值而改變(如下圖)，下圖中 type() 函數用於回傳變數資料類型。雖然變數資料類型會隨儲存值而改變，可是使用者有時會想要明確設定變數資料類型，這個時後可以使用接下來的變數資料類型來產生或轉換成對應類型。

變數資料類型

Python資料類型包含：文字類型(str)、數字類型(int、float、complex)、順序類型(list、tuple、range)、對應類型(dict)、集合類型(set、frozenset)、布林類型(bool)、二進制類型(bytes、bytearray、memoryview)，以下對於各個資料類型進行簡單描述。

下圖顯示當使用者對文字性數字進行運算時，Python會顯示顯示錯誤，Python + 號除了用於數字計算之外，如果對象為文字時， + 號用於文字合併。但是當使用者如果希望將文字性的整數 a 轉換為數字，則可使用資料類型 int 來進行轉換，浮點數與複數的轉換使用 float 與 complex。

順序類型(list、tuple、range)：

順序類型(list、tuple、range)用單一變數儲存一連串的資料，差異是在清單(list)將資料置於方括號([])中、元素(tuple)將資料置於小括號(())中、範圍(range)則為一整數序列。清單(list)項目內容可改變，有順序，使用方括號來建立清單，方括號搭配項目參考位置值來存取項目，第一個項目參考位置值為0，清單項目內容值可以不同。

元素(tuple)項目內容建立後不可改變，有順序，使用小括號建立元素，方括號搭配項目參考位置值來存取項目，第一個項目參考位置值為0，元素項目內容值可以不同，如果元素建立後改變內容會產生錯誤。

範圍(range)項目為順序整數值，有順序，內容建立後不可改變，使用range()函數來產生範圍項目，方括號搭配項目參考位置值來存取項目，第一個項目參考位置值為0。

對應類型(dict)：

對應類型(dict)將資料定義於大括號({})中，項目資料為許多 項目名稱:項目值 的組合，使用者可使用方括符搭配項目名稱來取得項目值，項目名稱不可重複，項目內容可改變，有順序。

集合類型(set、frozenset)：

集合類型(set)將資料定義於大括號({})中，項目資料不可重複，項目內容可增減，沒有順序(資料位置不代表順序)，使用者可以用迴圈搭配運算子 in 來取得或檢查集合項目。集合類型(frozenset)將可循環資料類型(list、tuple、dict、set)轉換為內容不可增減的集合物件。

二進制類型(bytes、bytearray、memoryview)：

bytes 為有順序、不可修改，使用方式為(bytes(資料,編碼方式))。中，bytearray 為有順序、可修改，使用方式同 bytes 。memoryview 讓使用者在程式執行過程中可以直接存取 bytes、bytearray 值，加快存取速度。
